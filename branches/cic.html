Cyber Security & Cloud Computing (CIC)

All about the CIC branch at CBIT

📌 What is CIC?

CIC is a unique blend of cybersecurity and cloud computing. It focuses on protecting systems, networks, and data while learning how cloud platforms like AWS and Azure operate and support modern IT infrastructure.

📚 Subjects by Semester

  • Computer Networks, Linux, Cryptography
  • Cloud Computing Fundamentals, Virtualization
  • Network Security, Ethical Hacking, DevOps
  • Cloud Platforms (AWS, Azure), Penetration Testing

🏢 Companies That Visit CBIT for CIC

  • Accenture, Infosys, Capgemini, Wipro
  • HCL, Palo Alto, Microsoft (Cloud Roles)

🛠 Focus Areas / Skills

  • Ethical Hacking & Cyber Laws
  • Cloud Services (AWS, Azure, GCP)
  • DevOps Tools: Docker, Kubernetes
  • Security Tools: Wireshark, Burp Suite

🎯 Job Roles

  • Cyber Security Analyst
  • Cloud Engineer
  • DevOps Engineer
  • Penetration Tester

❓ Common Misconceptions

  • “CIC is only about hacking” ❌ — It’s more about system security and cloud infrastructure
  • “You must be a tech wizard” ❌ — Learning step-by-step is enough
