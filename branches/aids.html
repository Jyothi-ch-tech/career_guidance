Artificial Intelligence & Data Science (AIDS)

All about the AIDS branch at CBIT

📌 What is AIDS?

AIDS focuses on both Artificial Intelligence and Data Science. While AIML emphasizes building intelligent systems, AIDS concentrates more on data handling, data pipelines, statistical modeling, and drawing insights from data.

📚 Subjects by Semester

  • Python, R Programming, Data Structures, SQL
  • Data Visualization, Statistics, Machine Learning
  • Big Data, Data Mining, Deep Learning, AI Ethics

🏢 Companies That Visit CBIT for AIDS

  • HSBC, Optum, ZS Associates, Infosys
  • Amazon, Fractal Analytics, Deloitte

🛠 Focus Areas / Skills

  • Data Preprocessing & Cleaning
  • ML Algorithms (Supervised/Unsupervised)
  • Tools: Python, R, Power BI, Tableau
  • Big Data (Hadoop, Spark)

🎯 Job Roles

  • Data Analyst
  • Data Scientist
  • ML Developer
  • Business Intelligence Engineer

❓ Common Misconceptions

  • “AIDS has no coding” ❌ — Coding is essential for data processing and ML models
  • “It’s only about Excel” ❌ — Tools like Python, SQL, Tableau, R are important
